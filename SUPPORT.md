## Support

Use the following channels depending on request type.

## Bug Reports

* Open a GitHub issue with reproduction steps, expected behavior, and actual behavior.
* Include environment details and screenshots when useful.

## Feature Requests

* Open a GitHub issue describing the problem, desired outcome, and any constraints.
* Link related issues or discussions.

## Security Reports

* Do not post security vulnerabilities publicly.
* Follow the process in `SECURITY.md`.

## General Questions

* Use GitHub discussions if enabled.
* If discussions are not enabled, open a GitHub issue with the `type:task` or `type:story` label as appropriate.
